# Twitter-Markup
트위터 마크업 챌린지

## 개발환경
- 구름IDE or VSCode

## 마크업 언어
- HTML5

## 개발목적
- HTML의 태그 활용과 Selectiong Elements 사용 방법 공부
- 시맨틱 마크업 

## 개발순서

1. Header
2. Global Navigation
3. Main
4. Tweet Form
5. Timeline
6. Tweet
7. Aside
8. Footer
9. End
